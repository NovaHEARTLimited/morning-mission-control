🧘 Regulation Tools
Astronaut Reset Toolkit
When the mission feels hard — use these to recharge and reset!
🌀
🚀
Rocket Breaths
1.Breathe IN for 4 counts
2.Hold for 2 counts
3.Breathe OUT for 4 counts
4.Repeat 3 times 🌟
☄️
Meteor Shake-Out
1.Stand up tall
2.Shake hands like meteors!
3.Shake feet, shake whole body
4.Stop — take a breath
🌌
Zero Gravity Stretch
1.Reach arms up high
2.Float slowly side to side
3.Roll shoulders back gently
4.Let arms float down slowly
🔟
10-Second Countdown
1.Close eyes if you like
2.Count slowly: 10… 9… 8…
3.Keep breathing calmly
4.Open eyes at 1 — ready!
💪
Push-the-Wall Reset
1.Stand facing a wall
2.Push both hands into it
3.Hold and breathe for 5 secs
4.Release — shake it out
🌟
Quiet Orbit Mode
1.Find a cosy quiet spot
2.Hug a cushion or blanket
3.Sit quietly for 2 minutes
4.Signal when ready to re-join

💬 Parent Guide
Mission Script Swaps
Small word changes make a big difference — try these this week!
📢
❌ Instead of...
"Hurry up! We're going to be late!"
→
✅ Try saying...
"Mission timer says we're approaching launch — next step!"
❌ Instead of...
"Focus! You're being so slow."
→
✅ Try saying...
"Looks like we drifted off course — let's return to Mission Control."
❌ Instead of...
"Why aren't you ready yet?!"
→
✅ Try saying...
"Your next mission task has been unlocked — let's check the board!"
❌ Instead of...
"You're in a bad mood again."
→
✅ Try saying...
"Your system looks low on fuel. Let's do a quick astronaut reset."
❌ Instead of...
"Stop crying — it's not a big deal."
→
✅ Try saying...
"That's a lot for one brain at once. Let's do one mission step together."
🚀 Core Mission Principle
"Connection before correction — when the mission feels hard, calm guidance keeps it on course."
